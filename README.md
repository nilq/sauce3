An engine you can use to make games in **MoonScript**. It's free, open-source, and works on **Windows**, **Mac OS X**, **Linux**, **Android**, **iOS** and **Ouya**.

### Installation

```
luarocks install --server=http://luarocks.org/dev sauce3
```
